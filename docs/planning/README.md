# Planning Docs Index

- `status.md`: current planning state and milestone status snapshot.
- `logs.md`: append-only execution log.
- `todo.md`: active risks/bugs/tasks.
- `todo.completed.md`: closed risks/bugs/tasks.
- `feedback.md`: planning feedback history and disposition.

- Completed milestone docs must be renamed to `milestone-<id>.completed.md` in the same change that marks the milestone completed.
